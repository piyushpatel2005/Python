# CSV module

CSV module provides various methods to parse csv files separated by any single character delimiter. 

You can find examples for CSV module from /examples/csv/ folder in the repo.

It provides `DictReader` and `DictWriter` which are useful for parsing csv files into lists or dictionaries.
